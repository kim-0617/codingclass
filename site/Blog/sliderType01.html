슬라이더 유형01

슬라이더 유형 01 입니다. 이미지가 휙휙 지나가는 유형입니다. 아직 스크립트를 안해서 움직이지는 않습니다.

html

html 구조입니다. 여기에는 이미지 splite 기법으로 image를 넣을거라서 화살표나 정지, 플레이 버튼 등에다가 ir 효과를 주었습니다. 그 외에는 이미지가 들어가는 slider__img, 그안에 적절한 텍스트를 기입하고 마무리해 줍니다.

css

css 부분 입니다. 메인 이미지를 넣고, 텍스트 영역을 적절히 배치하고, background-image로 하나하나 자잘한 이미지를 넣습니다.
