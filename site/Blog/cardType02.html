css

css 부분 입니다. card에 적절한 width 값과 flex로 정렬한 후, 적절한 간격을 배치하시면 됩니다. 그리고 더보기란은 background-image로 작업해서 position : absolute로 작업했습니다. 그런데 ir효과라는 개념이 처음 등장했습니다. 이것이 뭘까요?

ir 효과?

ir 효과란 img tag의 alt 속성을 작성하는 것과 비슷합니다. 웹표준을 지키기위해 이미지가 다운로드 되는 동안 페이지가 읽혔을 때, 적절한 텍스트를 일게 하는 것이죠. 더보기 란에 적절한 텍스트를 넣어주고 눈에는 보이지 않는 곳으로 치워버렸습니다. 

/* ir 효과 (이미지 대체 효과 === alt) */
