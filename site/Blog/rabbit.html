토끼 펜툴로 그리고 색칠하기

안녕하세요... 펜툴로 토끼를 그려서 Live paint로 색칠하는 과정을 글로 담았습니다. 토끼 아픈거 아닙니다. 결과를 봐주세요.

설명

1. 펜툴로 토끼 누끼를 정성스럽게 따줍니다. (눈은 원으로 그렸어요)
2. 누끼를 다른곳으로 옮겨서 object -> expand apperance 해줍니다.
3. object -> live paint -> k(단축키) 눌러서 색을 칠해주면 됩니다.
4. 명암을 넣을 땐 색칠한 후 다시 펜툴의 색을 지워서 영역만 잡고 조금 어두운 색으로 칠해주면 됩니다.
