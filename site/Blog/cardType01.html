카드 레이아웃 01

이번 레이아웃은 실제 사이트에서 많이 볼 수 있는 이미지와 설명이 함께있는 카드 유형의 레이아웃 입니다. 웹 표준을 위해 img태그를 figure태그와 같이 사용하였고 적절한 대체 텍스트를 입력하였습니다. 또한 시멘틱 태그에 적절한 제목도 추가하였습니다. 피그마로 먼저 레이아웃 작업을 한 뒤, 간격에 맞게 코딩만 하면 됩니다.

html

html 구조입니다. 섹션 부분에 제목과 소제목 부분, 표현하고자 하는 카드가 3개여서 article 3개로, 각각의 article은 (article header, article body) 구조입니다. article header에는 이미지를, article body에는 내용을 입력해줍니다.

css

css 부분 입니다. 카드 부분을 display:flex로 정렬해 주고, 피그마로 미리 만들어 두었던 간격과 font-size대로 css 작업을 해주면 끝입니다.

만약 글자 수가 많아져서 제목의 텍스트가 넘친다면?

white-space: nowrap;
text-overflow: ellipsis;
padding-right: 20px;
