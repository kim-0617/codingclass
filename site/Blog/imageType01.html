이미지 레이아웃 01

이번에 소개해드릴 유형은 이미지 레이아웃 입니다. 이미지 안에 텍스트가 있는 형태입니다. 간단하죠? 그림으로 먼저 보시겠습니다.

html

html 구조입니다. 간단합니다. 제목, 설명, 자세히보기가 image 영역안에 포함되어 있습니다. 

<section id="imageType01" class="image__wrap nexon section">
    <h2>강렬한 색감은 사람을 이끈다</h2>
    <p>이렇게 강렬한 사람을 본적 있습니까? 이게 바로 색의 효과입니다.  빨강색은 뭐든 돋보기에 해주죠</p>
    <div class="image__inner container">
        <article class="image img1">
            <h3 class="image__title">빨강은 강렬한 색채의 상징</h3>
            <p class="image__desc">
                빨강은 무척이나 강렬한 색채입니다. 
                색 그자체가 의미하는 바도 명확하구요. 
                이렇게 무채색 계열에 빨강을 입히면 주목성을 이끌 수 있습니다...
            </p>
            <a href="#" title = "자세히 보기" class="image__btn">자세히 보기</a>
        </article>
        <article class="image img2">
            <h3 class="image__title">쓸쓸한 느낌의 검정</h3>
            <p class="image__desc">
                그림에서 무엇이 연상되시나요? 
                고독.. 겨울.. 기약없는 기다림... 이러한 것들이 내포되어 있죠.. 
                검정색의 배경은 사람을 참 쓸쓸하게 하죠?
            </p>
            <a href="#" title = "자세히 보기" class="image__btn yellow">자세히 보기</a>
        </article>
    </div>
</section>

css

css 부분 입니다. image 부분을 display:flex로 정렬해 주고, 피그마로 미리 만들어 두었던 간격과 font-size대로 css 작업을 해줍니다. image클래스 내부에 css를 이용해서 (background) 배경이미지를 넣습니다. 

.image__inner{
    display: flex;
    justify-content: space-between;
}
.image{
    width: 49%;
    height: 370px;
    color: #fff;
    padding : 200px 30px 30px 30px;
    box-sizing: border-box;
}
.image.img1{
    background: url('./img/imgType01_01.jpg') no-repeat center;
}
.image.img2{
    background: url('./img/imgType01_02.jpg') no-repeat center;
}
.image__title{
    font-size: 32px;
    margin-bottom: 10px;
}
.image__desc{
    font-size: 16px;
    margin-bottom: 10px;
    line-height: 1.5;
    font-weight: 300;
    overflow: hidden;
    text-overflow: ellipsis;
    display: -webkit-box;
    -webkit-line-clamp: 2;
    -webkit-box-orient: vertical;
}
.image__btn{
    font-size: 16px;
    color: #fff;
    background-color: #99420F;
    padding: 10px 20px;
    display: inline-block;
}
.image__btn.yellow {
    background-color: #464444;
}
