배너 유형01

안녕하세요! 배너 유형01 입니다. 배너는 웹상의 온라인 광고의 일종입니다. 이미지와 간단한 텍스트가 들어갑니다.

html

html 구조입니다. banner__inner (전체 배너 영역) 부분에 이미지를 삽입하고, title과 desc를 작성해주면 코딩 끝입니다. 

<section id="bannerType" class="banner__wrap nexon">
    <h2 class="blind">배너 영역</h2>
    <div class="banner__inner">
        <h3 class="title">강렬한 색의 조화</h3>
        <p class="desc">
            더 많은 정보는 아래 링크를 참조해 주세요
            <a href="#">youtube.com/c/Webstoryboy</a>
        </p>
        <span class="small">배너 유형01</span>
    </div>
</section>

css

banner__inner 부분에 background-image로 배경이미지를 삽입하고, 제목과 본문을 적절히 배치해주면 끝입니다. 간단하죠? 

/* bannerType */
.banner__inner {
    background-image: url(../site1/asset/img/banner_bg01.jpg);
    background-repeat: no-repeat;
    background-position: center;
    background-size: cover;
    text-align: center;
    padding: 120px 0;
    color: #fff;
}

.banner__inner .title {
    font-size: 50px;
    line-height: 1;
    font-weight: 300;
    margin-bottom: 40px;
}

.banner__inner .desc {
    font-size: 24px;
    line-height: 1.5;
    font-weight: 300;
    margin-bottom: 70px;
}

.banner__inner .desc a {
    color: #fff;
    display: block;
}

.banner__inner .desc a:hover {
    text-decoration: underline;
}

.banner__inner .small {
    font-size: 16px;
    text-decoration: underline;
}
