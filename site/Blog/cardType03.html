카드 레이아웃 03

카드 레이아웃 03번 입니다. 카드는 3장입니다. 이번 유형은 카드를 감싸는 영역을 정의해서 카드와 카드사이의 여백을 일정한 간격을 줬고, 사이사이에 테두리 선이 있습니다. 가시성을 위해 배경색을 채웠는데요. 간격 설정하기가 까다로웠던것 같습니다. 색에 집중하기 보다는 이미지와 이미지사이의 간격이나 박스 배치에 신경써주세요.

html

html 구조입니다. 카드가 3개가 들어가죠? 각각의 카드는 header, contents, footer 3부분으로 나눴습니다. 그렇게 하면 구조가 복잡해져도 구역에 맞게 스타일을 줄 수 있겠죠?

css

css 부분입니다. card__header 부분에 img와 캡션 부분 처리를 해줬습니다. card__contents 부분에는 제목과 본문에 글자크기와 line-height를 설정해주면 됩니다. card__footer 부분에는 flex로 정렬하고 flex-end로 오른쪽 배치를 하였습니다. 그 외에는 간격을 맞춰서 스타일을 적용해 주면 됩니다.
