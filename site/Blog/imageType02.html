css

css 부분 입니다. 카드가 3개로 늘어난 만큼 간격에 변동만 있고 기본적인건 01번과 같습니다. 그리고 .desc 부분에 bottom : -100px, image영역에 overflow : hidden 작업을 해서 평소에는 보이지 않다가 image에 마우스를 hover 했을 때 bottom : 0 이 되어서 올라오는 효과처럼 보이게 돼요. transition : 적용할 애니메이션, 초, 효과주기 => 이 효과주기 부분에 ease-in을 주게되면 좀더 부드럽죠?
