CSS Animation Steps

안녕하세요, 오늘은 CSS Animation Steps 속성을 알아보겠습니다. steps의 지정한 값으로 gif 움직이는 이미지 파일을 만들 수 있습니다.

코드

html

위쪽 h3코드는 전체를 감싸는 폼 영역이구요, steps에 애니메이션을 주어서 걸어가는 효과를 주었습니다. 

<div class="sample">
    <h3><span></span><em class="sr-only"></em></h3>
    <div class="timing step">
        <div class="stepbox"></div>
        <span class="stepBtn">
            <a href="#" class="stepBtnStart">Start</a>
            <a href="#" class="stepBtnStop">Stop</a>
        </span>
    </div>
</div>

css

애니메이션 ani를 정의 합니다. 사진이 24프레임으로 가로로 길게 800px * 24장 해서 19200px 만큼 있는데 steps의 width가 800px 이니까 steps속성을 24로 주면 지정한 시간동안 24스텝으로 이미지가 이동할겁니다. (원래는 0%에서 0px ~ 100% -19200px 인데 24스텝으로 가니까요?)
