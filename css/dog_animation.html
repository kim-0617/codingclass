css

애니메이션 효과입니다. 전부 position을 absolute로 설정한 다음 위치를 맞춥니다. 사실 애니메이션은 대단할건 없습니다. keyframes "애니메이션 이름" 으로 주고, 0~100% 맞춰서 transform효과를 걸어주면 애니메이션 효과가 되겠죠? 코드는 SCSS로 보여드리겠습니다. 이건 Pub보다 살짝 어렵네요. 변수는 $로 선언해 줄 수 있습니다.

SCSS 코드 
$dog-width: 100px;
$interval: 200ms;
$color-gray: #EAEBEC;
$easing: ease-in-out;
body {
    display: flex;
    flex-direction: column;
    justify-content: center;
    align-items: center;
}
html, body {
    background: linear-gradient(90deg, rgba(2,0,36,1) 0%, rgba(255,0,0,1) 42%, rgba(121,9,9,1) 86%);
    width: 100%;
    height: 100%;
    margin: 0;
    padding: 0;
}
*, *:before, *:after {
    box-sizing: border-box;
    position: relative;
}
// dog
.dog {
    width: $dog-width;
    height: $dog-width;
    z-index: 1;
    
    &:before {
        content: '';
        display: block;
        position: absolute;
        width: 100%;
        height: 100%;
        border-radius: 50%;
        background: rgba(black, 0.03);
        transform: translatey(-30%) scale(1.5);
    }
    
    * {
        position: absolute;
    }
}
.dog-body, .dog-head, .dog-torso {
    border-radius: 50%;
    background: white;
    position: absolute;
    width: 100%;
    height: 100%;
}
// dog-body
.dog-body {
    z-index : -11;
    top: -100%;
    background-color: black;
    box-shadow : inset 0 -15px 0 0 $color-gray;
    animation: dog-body $interval $easing infinite alternate;
    
    &:before {
        content:'';
        position: absolute;
        bottom: 90%;
        right: 50%;
        width: 90%;
        height: 90%;
        background: rgba(gray, 0.4);
    border-top-left-radius: 100%;
    border-bottom-left-radius: 10%;
    border-top-right-radius: 10%;
    transform-origin: right bottom;
    animation : dog-tail-blur $interval $interval/6 $easing infinite alternate both;
    }
    @keyframes dog-body {
    from {transform: translatex(-10%)}
    to {transform: translatex(10%)}
    }
    @keyframes dog-tail-blur{
    0%{
        transform : rotate(0);
        opacity : 0;
    }
    50%{
        opacity :1;
    }
    100%{
        transform : rotate(90deg);
        opacity : 0;
    }
    }
}
// dog-head
.dog-head {
    background-color: #7d1817;
    animation: dog-head $interval * 9 cubic-bezier(0.11, 0.79, 0, 0.99) infinite;
