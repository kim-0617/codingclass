간단한 애니메이션 효과 03

안녕하세요. 텍스트가 순서대로 통통 튀는 효과를 만들어 볼게요!

코드

html

h1 태그 안에 글자 수 만큼 span태그를 만드시고 그안에 글자 한글자씩 입력합니다. 

<h1>
    <span>I</span>
    <span> </span>
    <span>a</span>
    <span>m</span>
    <span> </span>
    <span>L</span>
    <span>e</span>
    <span>g</span>
    <span>e</span>
    <span>n</span>
    <span>d</span>
  </h1>

css

글자를 화면 가운데 정렬하고 크기 조정 후, text-shadow를 주어서 글자에 그림자 효과를 줍니다. animation bounce를 정의해 두고 top 20px 이었던 글자를 -20px로 이동시키면서 글자가 튀는 효과를 줍니다. 글자 하나하나에 딜레이를 주어서 차례대로 통통 튀는 느낌을 줍니다.
