간단한 애니메이션 효과 02

안녕하세요. 바로 이어가 보겠습니다. 이번엔 원의 회전 효과 애니메이션인데요, 마치 지구가 태양을 돌듯 회전하는 효과입니다.

코드

html

원 두개가 돌겁니다. 그 밖에는 중심이 될 원입니다.

<div class = "loading">
    <span class= "circle1"></span>
    <span class= "circle2"></span>
</div>
<span class="circle3"></span>

css

애니메이션 간단하게 주면 됩니다. rotate(0) => rotate(360deg) 이렇게 무한히 반복하면 원이 loading을 중심으로 계속 돌게 되겠죠? 

body{
    height: 100vh;
    background-image : linear-gradient(-60deg, #ff5858 0%, #f09819 100%);
}

.loading{
    position: absolute;
    left: 50%;
    top: 50%;
    width: 20px;
    height: 150px;
    animation : loading 0.5s linear infinite;
}

.circle3{
    display: block;
    width: 40px;
    height: 40px;
    border-radius: 50%;
    background: red;
    position: absolute;
    top: 45%;
    left: 48%;
}

.loading .circle1{
    display: block;
    width: 20px;
    height: 20px;
    border-radius: 50%;
    background: #000;
    margin-bottom: 40px;
}

.loading .circle2{
    display: block;
    margin-top: 110px;
    width: 20px;
    height: 20px;
    border-radius: 50%;
    background: #000;
}

@keyframes loading{
    0%{transform : translate(-50%, -50%) rotate(0deg);}
    100%{transform : translate(-50%, -50%) rotate(360deg);}
}
