Javascript

  • 01. number.MAX_VALUE : 나타낼 수 있는 최대 숫자 값을 나타냅니다.
  • 02. number.MIN_VALUE : 나타낼 수 있는 최소 숫자 값을 나타냅니다.
  • 03. number.MAX_SAFE_INTEGER : 최대 안전 정수(2 – 1)를 나타냅니다.
  • 04. number.MIN_SAFE_INTEGER : 최소 안전 정수 또는 -(2 - 1)를 나타냅니다.
  • 05. number.Nan : Not-A-Number를 나타냅니다.
  • 06. number.isFinite : 전달된 값이 유한수인지 여부를 판단합니다.
  • 07. number.isInteger : 메서드에 따라 전달된 값이 정수인지 여부가 결정됩니다.
  • 08. number.isNaN : 강제로 매개변수를 숫자로 변환하지 않고, 주어진 값의 유형이 Number이고 값이 NaN이면 true, 아니면 false 반환합니다.
  • 09. number.prototype.toFixed : 숫자를 고정 소수점 표기법(fixed-point notation)으로 표시합니다.
  • 10. number.prototype.toPrecision : Number 객체를 지정된 정밀도로 나타내는 문자열을 반환한다.
  • 11. number.prototype.toString : 특정한 Number 객체를 나타내는 문자열을 반환합니다.


  • 01. new Date : 시간의 특정 지점을 나타내는 Date 객체를 플랫폼에 종속되지 않는 형태로 생성
  • 02. date.now : UTC 기준으로 1970년 1월 1일 0시 0분 0초부터 현재까지 경과된 밀리 초를 반환합니다.
  • 03. date.parse : 날짜의 문자열 표현을 구문 분석하고 1970 년 1 월 1 일 00:00:00 UTC 이후의 밀리 초 수를 반환하거나 문자열이 인식되지 않거나 경우에 따라 잘못된 날짜 값을 포함하는 경우 NaN을 반환합니다.
  • 04. date.UTC : Date 생성자와 비슷한 매개변수를 받지만, 모두 UTC로 취급합니다. 반환 값은 1970년 1월 1일 00:00:00 UTC부터 매개변수가 나타내는 시간의 차이를 밀리초로 나타낸 수입니다.
  • 05. date.prototype.getFullYear : 주어진 날짜의 현지 시간 기준 연도를 반환합니다.
  • 06. date.prototype.setFullYear : 현지 시간에 따라 지정된 날짜의 전체 연도를 설정합니다.
  • 07. date.prototype.getMonth : Date 객체의 월 값을 현지 시간에 맞춰 반환합니다. 월은 0부터 시작합니다.
  • 08. date.prototype.setMonth : 현재 설정된 연도에 따라 지정된 날짜의 월을 설정합니다.
  • 09. date.prototype.getDate : 주어진 날짜의 현지 시간 기준 일을 반환합니다.
  • 10. date.prototype.setDate : 현재 설정된 월의 시작 부분을 기준으로 Date 객체의 날짜를 설정합니다.
  • 11. date.prototype.getDay : 주어진 날짜의 현지 시간 기준 요일을 반환합니다. 0은 일요일을 나타냅니다.
  • 12. date.prototype.getHours : 주어진 날짜의 현지 시간 기준 시를 반환합니다.
  • 13. date.prototype.setHours : 현지 시간에 따라 지정된 날짜의 시간을 설정하고 1970 년 1 월 1 일 00:00:00 UTC 이후 업데이트 된 Date에 의해 표시되는 시간 (밀리 초)을 반환합니다.
  • 14. date.prototype.getMinutes : Date 인스턴스의 분을 현지 시간 기준으로 반환합니다.
  • 15. date.prototype.setMinutes : 현지 시간에 따라 지정된 날짜의 분을 설정합니다.
  • 16. date.prototype.getSeconds : Date 객체의 초 값을 현지 시간에 맞춰 반환합니다.
  • 17. date.prototype.setSeconds : 현지 시간에 따라 지정된 날짜의 초를 설정합니다.
  • 18. date.prototype.getMilliseconds : Date 인스턴스의 밀리초를 현지 시간 기준으로 반환합니다.
  • 19. date.prototype.setMilliseconds : 현지 시간에 따라 지정된 날짜의 밀리 초를 설정합니다.
  • 20. date.prototype.getTime : 표준시에 따라 지정된 날짜의 시간에 해당하는 숫자 값을 반환합니다.
  • 21. date.prototype.setTime : Date 객체를 1970 년 1 월 1 일 00:00:00 UTC부터 밀리 초 단위로 나타내는 시간으로 설정합니다.
  • 22. date.prototype.getTimezoneOffset : 현재 로케일 (즉, 호스트 시스템 설정)에 대한 시간대 오프셋 (UTC)을 분 단위로 반환합니다.
  • 23. date.prototype.toDateString : 미국사람이 읽을 수있는 형태로 Date 객체의 날짜 부분을 반환합니다.
  • 24. date.prototype.toTimeString : 로컬 시간대로 해석된 객체의 시간 부분을 영어로 반환합니다.
  • 25. date.prototype.toLocaleString : 날짜를 언어별로 나타내는 문자열을 반환합니다
  • 26. date.prototype.toLocaleTimeString : 날짜의 시간 부분을 언어별로 나타내는 문자열을 반환합니다.
