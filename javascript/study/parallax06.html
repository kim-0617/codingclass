추가된 부분입니다. split에 show가 붙었을 때 span에 애니메이션이 나와야 합니다. 그래서 그전에는 opacity를 0으로, show가 붙으면 opacity를 1로 만들어주는 셋팅을 해줍니다.

코드 보기 
/* option */
.split span{
    display: inline-block;
    opacity: 0;
    transform: translateY(50px);
    transition: all 0.3s cubic-bezier(0.79, -1.82, 0.33, 3.07);
    min-width: 1vw;
    min-height: 1vw;
}
.split.show span {
    opacity: 1;
    transform: translateY(0px);
}

js

순서를 짚어보죠.. 1. 글씨를 쪼개야 합니다. 클래스 split이 붙은 요소만요 그러기 위해서 split 클래스를 모두 선택해줍니다.
2. 텍스트를 선택해서 split합니다. 그렇다면 한글자 한글자가 배열에 원소로써 담기게 되겠죠? 그걸 다시 join 할건데..
3. 구분자로서 span태그로 감싸줍니다. 그러면 하나하나 요소가 span태그로 감싸지겠죠? 그런데 여기서 잠깐..
4. <span></span>로 감싸주게 되면 짝이 안맞습니다. 이건 설명보다 직접 해보는게 빠릅니다. 그래서 닫는 태그를 먼저 써준겁니다.
5. 이렇게되면 첫부분의 시작태그와 제일 끝부분의 종료태그가 없는데 이건 직접 더해주도록 합시다.
6. 그리고 이렇게 만들어진 span태그 들을 다시 p태그에 넣어줍니다. 그럼 애니메이션 줄 준비가 끝났어요.
7. 섹션을 잡아줍니다(9개) 여기에 offsetTop값과 scrollTop 값을 비교해서 해당 섹션 근처에 도달했을 때, show 클래스를 붙여줍니다.
8. 또 해줘야 할게 있는데, split 클래스의 span 모두 선택해서 각각의 span에다가 transition-delay를 줘야합니다. 그래야 한글자씩 나타나겠죠?
9. span.style.transitionDelay = `${index * 50}ms`; 요렇게, 글자마다 0.05초씩 딜레이를 주면 완성입니다.
10. 아참, window.addEventListener('scroll')보다 requestAnimationFrame으로 scroll event를 구현하는게 더 좋다고 했죠?

코드 보기 
document.querySelectorAll('.split').forEach((text) => {
    let splitText = text.innerText;
    let splitWrap = splitText.split('').join('</span><span aria-hidden = "true">');
    splitWrap = "<span aria-hidden = 'true'>" + splitWrap + "</span>";
    text.innerHTML = splitWrap;
    text.setAttribute("aria-label", splitText);
});

function scroll() {
    let scrollTop = window.scrollY;
    document.querySelector('.scroll span').innerText = Math.ceil(scrollTop);
 
    document.querySelectorAll('.content__item').forEach((item) => {
        if(scrollTop >= item.offsetTop - 200) {
            item.querySelector('.split').classList.add('show');
            item.querySelectorAll('.split span').forEach((span, index) => {
                span.style.transitionDelay = `${index * 50}ms`;
            });
        }
    });
    requestAnimationFrame(scroll);
}
