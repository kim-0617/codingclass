js

자 js 부분인데요? 순서대로 가보겠습니다.
1. 현재 스크롤이 얼마인지는 let scrollTop = window.pageYOffset || window.scrollY || document.documentElement.scrollTop; 이용해서 구합니다.
2. 그걸 document.querySelector('#parallax__info .scroll span') 여기다 넣어줬어요
3. 각 섹션의 시작 값 (즉, 얼마만큼 내려야 각 섹션을 만날수가 있느냐)은 document.getElementById("section" + i).offsetTop
4. 즉, section(1,2,3 ...) 의 offsetTop 속성으로 구할수가 있습니다. 그것을 document.querySelector(".offset" + i) 여기다 넣어줬죠?
5. 자 그래서 각 메뉴에 active는 어떻게 구하느냐? document.querySelectorAll('.content__item').forEach 이부분은 section 9개를 집었죠?
6. if (scrollTop >= element.offsetTop - 1) => 현재 내린 스크롤 값이 섹션의 offsetTop값 보다 커지면, 즉 더 많이 내리면 active를 현재 섹션으로 바꿉니다.
7. 나머지 섹션에는 active를 지워줘야 겠죠? 그렇게해서 active를 바꿔주시면 되고,
8. 마지막으로 smooth 효과를 집어보죠. document.querySelectorAll('#parallax__nav li a') => 우리가 메뉴를 클릭했을 때 이동하는 부분이죠? (각각의 섹션 id 로요)
9. 여기서 a링크를 클릭했을 때 addEventListener("click") 이벤트를 설치하고, a값의 href 속성을 얻어온 후, 대망의 scrollIntoView 메서드를 사용하겠습니다.
10.scrollIntoView는 href 속성으로 이동할 때의 효과를 지정할 수 있습니다. 인자로 객체를 넘기는데, behavior : "smooth",를 넘겨주면 스무스하게 넘어가죠?
