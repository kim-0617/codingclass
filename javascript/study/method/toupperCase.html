toUpperCase() / toLowerCase()

이번 시간도 쉽습니다. toUpperCase는 문자열을 모두 대문자로, toLowerCase는 문자열을 모두 소문자로 만듭니다.

toUpperCase() 메서드
str.toUpperCase(); // 문자열을 대문자로 바꿈
toLowerCase() 메서드
str.toLowerCase(); // 문자열을 소문자로 바꿈

toUpperCase() 소개

str.toUpperCase() 메서드란 str이 대문자로 바꾼 값을 리턴합니다.

const str1 = 'javascript';
const currentStr1 = str1.toUpperCase();
결과 확인하기
JAVASCRIPT

toLowerCase() 소개

str.toLowerCase() 메서드란 str이 소문자로 바꾼 값을 리턴합니다.

const str2 = 'JAVASCRIPT';
const currentStr2 = str2.toLowerCase();
