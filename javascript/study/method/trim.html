trim() / trimStart() / trimeEnd()

이번 메서드는 trim 입니다. 앞에 start와 end가 붙은건 무슨뜻인지 아시겠죠, 앞/뒤 공백만 선택적으로 제거할 때 사용합니다.

trim() 메서드
str.trim()); // 문자열의 앞/뒤 공백을 제거합니다. str.trimStart()); // 문자열의 앞 공백을 제거합니다. str.trimEnd()); // 문자열의 뒤 공백을 제거합니다.

trim() 소개

str.indexOf("문자열") 메서드란 str안에 "문자열"이 포함되어 있으면 그 문자열의 시작위치를 반환한다. 없으면 -1을 반환하죠

const str1 = "     javascript       ";
const currentStr = str.trim();

const str2 = "     javascript       ";
const currentStr2 = str2.trimStart();

const str3 = "     javascript       ";
