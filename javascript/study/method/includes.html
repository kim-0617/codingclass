includes()

includes 메서드입니다. searchEffect01 에서는 indexOf를 이용한 검색기능을 구현해보았죠? 하지만 그건 첫번째 글자만 인식했어요? 왜그러느냐, 패턴에 맞는 문자 한번만 찾기 때문이죠 하지만 이건 포함만 하면 찾아버리기 때문에 좀 더 유연한 검색 기능을 구현할 수 있겠죠?

includes() 메서드
"문자열".includes(검색값)
// 문자열내부에 검색값이 있다면 true, 그렇지 않다면 false를 반환합니다 "문자열".includes(검색값, 시작값) // 시작값을 주면 시작값부터 탐색합니다.

includes() 소개

str1.includes() 메서드는 인자로 들어온 문자열이 str1문자열 내에 포함되어 있으면 true, 그렇지 않다면 false를 반환합니다. 조건문에서 많이 쓰이겠죠? 원리는 간단하죠? 


const str1 = 'javascript reference';
const currentStr1 = str1.includes('javascript'); // true
const currentStr2 = str1.includes('j'); // true
const currentStr3 = str1.includes('b'); // false
const currentStr4 = str1.includes('reference'); // true
const currentStr5 = str1.includes('reference', 1); // true
const currentStr6 = str1.includes('reference', 11); // true
const currentStr7 = str1.includes('reference', 12); // false
