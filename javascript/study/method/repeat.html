repeat()

이번시간에는 간단한 메서드 소개시켜드릴게요. repeat()는 반복하다라는 뜻이죠? 인자로 들어온 문자열을 반복시켜서 반환해줍니다.

repeat() 메서드
str.repeat("반복횟수"); // 반복횟수만큼 str 출력

repeat() 소개

앞서 말했듯이 "문자열".repeat("반복 횟수") 하면 문자열을 반복횟수만큼 반복하는 겁니다. 그게 다에요. 별찍기 할 때 써보세요 

const str1 = "javascript";
const currentStr1 = str1.repeat(0); // ''
const currentStr2 = str1.repeat(1); // 'javascript'
const currentStr3 = str1.repeat(2); // 'javascriptjavascript'
결과 확인하기
''
'javascript'
'javascriptjavascript'
