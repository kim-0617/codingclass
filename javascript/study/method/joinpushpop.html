배열의 여러가지 메서드!

배열의 원소를 조작할 수 있는 다양한 메서드들이 있습니다. 지금부터 함께 보시죠.. querySelector부분은 신경쓰지 말아주세요. 중요한건 메서드들이 무슨 기능을 하느냐 입니다. 여러분들의 성원에 힘입어 내용을 추가했습니다.

join() 메서드
array.join("구분자");

push() 메서드
array.push("배열에 추가할 원소");

pop() 메서드
array.pop();

다양한 방법으로 join 활용해보기

배열의 join메서드는 구분자의 형태에 따라 다양한 형태로 배열을 문자열로 결합시켜 줍니다.

const arrNum = [100, 200, 300, 400, 500];
const text1 = arrNum.join(""); // 배열을 공백없이 문자열로 결합합니다.
const text2 = arrNum.join(" "); // 배열을 한칸띄고 문자열로 결합합니다.
const text3 = arrNum.join("*"); // 구분자를 '*' 로 해서 배열을 문자열로 결합합니다.
const text4 = arrNum.join("-"); // 구분자를 '-' 로 해서 배열을 문자열로 결합합니다.
결과 확인하기
100200300400500
100 200 300 400 500
100*200*300*400*500
100-200-300-400-500

push 기능 알아보기

array.push("원소");는 배열에 push의 인자로 들어온 원소를 맨 뒤에 추가합니다. 반환 값은 추가된 배열의 길이입니다.

const arrNum = [100, 200, 300, 400, 500];
const arrPush = arrNum.push(600); // 배열에 600을 추가하고 새로운 배열의 길이를 반환합니다.

document.querySelector(".sample03_N1").innerHTML = "1";
document.querySelector(".sample03_Q1").innerHTML = "[100, 200, 300, 400, 500]";
document.querySelector(".sample03_M1").innerHTML = "arrNum.push(600)";
document.querySelector(".sample03_P1").innerHTML = arrPush;
document.querySelector(".sample03_A1").innerHTML = arrNum;
결과 확인하기
배열 : [100, 200, 300, 400, 500, 600]
리턴 값 : 6

pop 기능 알아보기

array.pop();은 배열에서 마지막 요소를 제거한뒤 제거한 요소를 반환합니다.

const arrNum2 = [100, 200, 300, 400, 500];
const arrPush2 = arrNum2.pop(); // 배열의 마지막 요소를 삭제합니다.

document.querySelector(".sample03_N2").innerHTML = "2";
document.querySelector(".sample03_Q2").innerHTML = "[100, 200, 300, 400, 500]";
document.querySelector(".sample03_M2").innerHTML = "arrNum.pop()";
document.querySelector(".sample03_P2").innerHTML = arrPush2;
document.querySelector(".sample03_A2").innerHTML = arrNum2;
결과 확인하기
배열 : [100, 200, 300, 400]
반환 값 : 500

push, pop 이용해보기

push와 pop 메서드를 이용해서 배열을 뒤집어볼게요!

const arr = [1,2,3,4,5,6,7];
const reversed_arr = [];
while(arr.length !== 0){
    reversed_arr.push(arr.pop());
}
console.log(reversed_arr);
결과 확인하기
reversed_arr : [7,6,5,4,3,2,1]
