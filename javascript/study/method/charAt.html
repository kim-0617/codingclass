charAt()

안녕하세요, 이번시간에도 문자열 메서드를 소개해드리겠습니다. 바로 charAt() 입니다.

charAt() 메서드
str.charAt("문자열의 위치값"); // 문자열에서 인자로 입력한 위치의 해당하는 문자를 반환합니다.

charAt() 배워보기

charAt은 문자열에서 인자로 들어온 숫자의 위치의 문자열은 무엇인가 되돌려줍니다.

const str1 = "javascript reference"; 
const currentStr1 = str1.charAt(); // j
const currentStr2 = str1.charAt("0"); // j
const currentStr3 = str1.charAt("1"); // a
const currentStr4 = str1.charAt("2"); // v
