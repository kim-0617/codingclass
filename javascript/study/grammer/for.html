반복문

반복문은 프로그램에서 필요한 결과 값을 도출하기 위해 실행문의 순서를 반복적으로 실행시키는 문법을 말합니다. 반복문에는 while문, do while문, for문이 있으며, for문을 가장 많이 사용합니다.

while문

while문은 조건식을 판단해서 참인경우 실행한다.

while (조건식) {
    실행문
}

do while문

do while문은 조건식이 어떻든 일단 한번 실행하고 조건문을 비교한다.

for (초기값; 조건식; 증감값) {
    실행문
}

for문

for문은 시작값, 조건문, 증감값을 일목요연하게 기입하여 반복문 안의 실행문을 실행한다. 가장 많이 사용하는 반복문

for (초기값; 조건식; 증감값) {
    실행문
}

break문

반목문에서 break 키워드가 등장하면, 그 즉시 반복문을 탈출한다.

for (초기값; 조건식; 증감값) {
    실행문
    if(조건) break; //탈출
}

continue문

반복문에서 continue 키워드가 등장하면, 그 밑부분을 건너뛰고 다음반복으로 넘어간다.

for (초기값; 조건식; 증감값) {
    실행문
    if(조건) continue; // 밑부분 스킵
    실행문
}

0부터 99까지 출력하기

i가 0부터 시작해서 1씩 증가되면서 99가 될 때까지 실행하며, i의 최종 값은 100입니다.

for ( let i = 0; i < 100; i++ ){
    document.write(i); // 0부터 99까지 출력하기
}

1부터 100까지 짝수만 출력하기

1부터 100까지 (5의배수)

5의 배수만 출력하려면 i값이 5로 나누어 떨어지는 것만 출력합니다.

for ( let i = 1; i <= 100; i++ ){
    if(i % 5 == 0){
        document.write(i); // 1부터 100까지 출력하기 (5의 배수)
    }
}

1부터 100까지 (5의배수는 빨간색 7의배수는 파란색)

5의 배수는 빨간색, 7의 배수는 파란색으로 출력해 보겠습니다.

for(let i = 1; i <= 100; i++){
    if(i % 5 == 0){
        document.write("<span style='color:red;'>"+ i +"</span>");
    }
    if(i % 7 == 0){
        document.write("<span style='color:blue;'>"+ i +"</span>");
    }
}

구구단

중첩 for문을 이용해서 구구단을 보여드리겠습니다..

for(let i = 2; i <= 9; i++){
    for(let j = 1; j <= 9; j++){
        document.write(i + "*" + j + "=" + i*j);

}

for문을 이용해서 테이블 출력하기

중첩 for문을 이용해서 테이블을 만들 수 있습니다.

// 100칸 출력합니다.
let table = '<table>';

for (let i = 0; i < 10; i++){
    table += "<tr>";
    for (let j = 0; j < 10; j++) {
        table += "<td>" + "★" + "</td>";
    }
    table += "</tr>";
}
table += "</table>"
document.write(table);

break문

break문을 이용해서 10까지만 출력할 수 있습니다.

for(let i = 1; i <= 20; i++){
    document.write(i);
    if(i === 10) break; // 10까지만 출력하고 반복문 탈출
}

continue문

continue문을 이용해서 10은 건너뛰고 20까지 출력해 보겠습니다.

for(let i = 1; i <= 20; i++){
    if(i === 10) continue; // 10은 건너뛴다. (밑에는 스킵된다.)
    document.write(i);
}
