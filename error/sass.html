설치 02

css 파일을 다른곳에 저장시키고 싶을 때.

1. settings.json파일에 다음의 코드를 추가 시켜줍니다.
2. "liveSassCompile.settings.generateMap": false, => map파일 생성안하기
3. 압축파일 생성 "extensionName": ".min.css", => 확장자 명
4. 저장경로 : "savePath": "~/../css/" => ~ : 현재 scss 파일이 있는 경로 

"liveSassCompile.settings.generateMap": false,
"liveSassCompile.settings.formats": [
    // {
    // "format": "expanded",
    // "extensionName": ".css",
    // "savePath": "../css/"
    // },
    {
    "format": "compressed",
    "extensionName": ".min.css",
    "savePath": "~/../css/"
    }
]
