레이아웃02

이번 레이아웃은 저번 시간에 배웠던 레이아웃보다 한 단계 진화해서, 순서대로 header, nav, (aside, section, article), footer로 구성되어 있습니다. css의 주석은 저번시간에 했기 때문에 생략하겠습니다.

01. float을 이용한 레이아웃

aside 영역과 section 영역과 article 영역을 한줄에 배치하기 위해서 float속성을 사용하겠습니다. 하지만 float의 영향 때문에 밑의 footer영역이 없어지기 때문에 clear:both 속성을 사용해서 float에서 자유롭게 해줍니다. 

* {
    margin: 0;
  }
  
  body {
    background-color: #E8F5E9;
  }
  
  #wrap {
    width: 1200px;
    margin: 0 auto;
    /* 블록구조를 가운데 정렬 */
  }
  
  #header {
    width: 100%;
    height: 100px;
    background-color: #C8E6C9;
    grid-area: header;
  }
  
  #nav {
    width: 100%;
    height: 100px;
    background-color: #A5D6A7;
    grid-area: nav;
  }
  
  #aside {
    width: 25%;
    height: 780px;
    background-color: #81C784;
    grid-area: aside;
    float: left;
  }
  
  #section {
    width: 50%;
    height: 780px;
    background-color: #66BB6A;
    grid-area: section;
    float: left;
  }
  
  #article {
    width: 25%;
    height: 780px;
    background-color: #4CAF50;
    float: left;
  }
  
  #footer {
    width: 100%;
    height: 100px;
    background-color: #43A047;
    grid-area: footer;
    clear: both;
  }
  
  @media(max-width : 1220px) {
    #wrap {
      width: 96%;
    }
  }
  
  @media(max-width : 768px) {
    #wrap {
      width: 100%;
    }
    #aside {
      width: 45%;
      height: 630px;
    }
  
    #section {
      width: 55%;
      height: 630px;
    }
  
    #article {
      width: 100%;
      height: 150px;
    }
  }
  
  @media(max-width : 480px) {
    #aside {
      width: 100%;
      height: 200px;
    }
  
    #section {
      width: 100%;
      height: 430px;
    }
  
    #article {
      width: 100%;
      height: 150px;
    }
  }

02. flex를 이용한 레이아웃

flex속성은 모든 태그의 부모인 wrap에 display:flex를 지정 후, 요소들이 잘 배치될 수 있도록 flex-wrap : wrap;를 사용해주면 간단하게 레이아웃이 완성됩니다. 

* {
    margin: 0;
}

body {
    background-color: #E8F5E9;
}

#wrap {
    width: 1200px;
    margin: 0 auto;
    /* 블록구조를 가운데 정렬 */
    display: flex;
    flex-wrap: wrap;
}

#header {
    width: 100%;
    height: 100px;
    background-color: #C8E6C9;
}

#nav {
    width: 100%;
    height: 100px;
    background-color: #A5D6A7;
}

#aside {
    width: 25%;
    height: 780px;
    background-color: #81C784;
}

#section {
    width: 50%;
    height: 780px;
    background-color: #66BB6A;
}

#article {
    width: 25%;
    height: 780px;
    background-color: #4CAF50;
}

#footer {
    width: 100%;
    height: 100px;
    background-color: #43A047;
}

@media (max-width : 1220px){
    #wrap{
        width: 96%;
    }
}
@media (max-width : 768px){
    #wrap{
        width: 100%;
    }
    #section{
        width: 60%;
        height: 630px;
    }
    #aside{
        width: 40%;
        height: 630px;
    }
    #article{
        width: 100%;
        height: 150px;
    }
}
@media (max-width : 480px){
    #section{
        width: 100%;
        height: 200px;
    }
    #aside{
        width: 100%;
        height: 430px;
    }
    #article{
        width: 100%;
        height: 150px;
    }
}

03. grid를 이용한 레이아웃

grid속성은 모든 태그들을 감싸는 부모태그에서 자식요소의 모든 것을 컨트롤 하는 것 입니다. 부모태그에서 grid-templates-area로 영역을 지정 후 grid-templates-row & column으로 열너비와 행높이를 지정합니다. 또한, 각각 자식태그에서 grid-templates-area로정의한 영역의 이름을 gird-area로 정의해 주어야 합니다. 

 * {
    margin: 0;
}

body {
    background-color: #E8F5E9;
}

#wrap {
    width: 1200px;
    margin: 0 auto;
    /* 블록구조를 가운데 정렬 */
    display: grid;
    grid-template-areas:
        "header header header"
        "nav nav nav"
        "aside section article"
        "footer footer footer"
    ;
    grid-template-columns: 25% 50% 25%;
    grid-template-rows: 100px 100px 780px 100px;
}

#header {
    background-color: #C8E6C9;
    grid-area: header;
}

#nav {
    background-color: #A5D6A7;
    grid-area: nav;
}

#aside {
    background-color: #81C784;
    grid-area: aside;
}

#section {
    background-color: #66BB6A;
    grid-area: section;
}

#article {
    background-color: #4CAF50;
    grid-area: article;
}

#footer {
    background-color: #43A047;
    grid-area: footer;
}

@media(max-width : 1220px) {
    #wrap{
        width: 96%;
    }
}

@media(max-width : 768px) {
    #wrap{
        width: 100%;
        grid-template-areas:
        "header header"
        "nav nav"
        "aside section"
        "article article"
        "footer footer"
    ;
    grid-template-columns: 40% 60%;
    grid-template-rows: 100px 100px 630px 150px 100px;
    }
}

@media(max-width : 480px) {
    #wrap{
        grid-template-areas:
        "header header header"
        "nav nav nav"
        "aside aside aside"
        "section section section"
        "article article article"
        "footer footer footer"
    ;
    grid-template-columns: 25% 50% 25%;
    grid-template-rows: 100px 100px 200px 430px 150px 100px;
    }
}
