레이아웃05

레이아웃 5번으로 돌아왔습니다. 이런 레이아웃은 웹디자인 기능사에서도 출제가 되는 디자인입니다. 헤더영역(2부분) / 광고배너영역 / 섹션영역(3부분) / 푸터영역(3부분) 로 구성되는 레이아웃 입니다.

float을 이용한 레이아웃

3부분으로 나뉘는 부분에 float 속성을 부여합니다. 그리고 부모요소에 overflow : hidden 속성을 주면 레이아웃을 깔끔히 배치할 수 있습니다. 텍스트 정렬은 text-align : center 속성과 line-height : 요소의 높이 만큼 주면 간단히 정렬할 수 있습니다. 

*{
    margin: 0;
    padding: 0;
    font-size: 30px;
}
#wrap{
    width: 1200px;
    margin: 0 auto;
}
#header{
    width: 1200px;
    height: 100px;
    background-color: #888;
    overflow: hidden;
    text-align: center;
    /* 글씨가 한줄일 때 line-height와 height를 같게 */
    line-height: 100px;  
}
#header .logo{
    width: 300px;
    height: 100px;
    background-color: #222;
    float: left;
    color: #fff;
}
#header .menu{
