레이아웃06

레이아웃 6번도 바로 가겠습니다. 이번 디자인 역시 웹디자인 기능사에서도 출제가 되는 디자인입니다. 헤더영역(2부분) / 광고배너영역(2부분) / 섹션영역(3부분) / 푸터영역(3부분) 로 구성되는 레이아웃 입니다. 각각의 요소마다 레이아웃을 적용해주어야겠네요.

float을 이용한 레이아웃

모든 부분에 float 속성을 부여합니다. 그리고 부모요소에 overflow : hidden 속성을 주면 레이아웃을 깔끔히 배치할 수 있습니다. 텍스트 정렬은 text-align : center 속성과 line-height : 요소의 높이 만큼 주면 간단히 정렬할 수 있습니다. 

*{
    margin: 0;
}
#wrap{
    width: 100%;
}
#header{
    width: 100%;
    height: 100px;
    background-color: #F48080;
    text-align: center;
    line-height: 100px;
}
#banner{
    width: 100%;
    height: 400px;
    background-color: #EC5C5C;
    text-align: center;
    line-height: 400px;
}
#ad{
    width: 100%;
    height: 500px;
    background-color: #FEA5A5;
    text-align: center;
    line-height: 250px;
}
#notice{
    width: 100%;
    height: 500px;
    background-color: #FFBFBF;
    text-align: center;
    line-height: 500px;
}
#footer{
    width: 100%;
    height: 100px;
    background-color: #FF9C9C;
    text-align: center;
    line-height: 100px;
}
.container{
    width: 1200px;
    height: inherit;
    margin: 0 auto;
    background-color: rgba(255, 255, 255, 0.3);
    overflow: hidden;
}
#header .logo{
    width: 300px;
    height: 100px;

    background-color: #F86464;

#header .menu{

}
