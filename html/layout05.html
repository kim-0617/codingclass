레이아웃05

레이아웃 5번으로 돌아왔습니다. 이번 레이아웃 역시 전체영역이 들어간 페이지에 container클래스를 가운데 정렬하여 레이아웃을 구성했습니다. 각 기기마다 section 부분의 레이아웃 방식이 달라지니 집중해서 봐주세요.

float을 이용한 레이아웃

float을 적용하기에 앞서 float으로 인한 영역깨짐 방지법을 알아보겠습니다.
/* float으로 인한 영역깨짐 방지법
1. 깨지는 영역에 clear:both를 설정한다.
2. 부모 박스 영역에 overflow : hidden을 설정한다.
3. clearfix를 설정한다.
*/
1번 방법은 깨지는 영역을 눈으로 확인할 수 있어야 적용할 수 있습니다.
2번 방법은 육안상으로는 잘 정렬된 것 처럼 보이지만, 영역 밖에서 안으로 들어오는 애니메이션 효과 같은게 있을 때 제대로 표시되지 않을 수 있습니다.
3번 방법이 가장 깔끔한 방법입니다. clearfix클래스를 적용해서 가상요소 앞뒤로 속성을 적용해 주고 float을 적용한 요소의 부모박스에 clearfix 클래스를 부여하면 됩니다.

    margin: 0;
}

    width: 50%;
    height: 480px;
    float: left;
    background-color: #4A352F;
}

/* float으로 인한 영역깨짐 방지법
    1. 깨지는 영역에 clear:both를 설정한다.
    2. 부모 박스 영역에 overflow : hidden을 설정한다.
    3. clearfix를 설정한다.
*/
.clearfix::before,
.clearfix::after{
    content : '';
    line-height: 0;
    display : block;
}
.clearfix::after{
    clear: both;
}

flex을 이용한 레이아웃

flex를 이용한 레이아웃 방법입니다.
flex를 적용하기 위해 float와는 조금다른 html구조를 사용했습니다. float처럼 형제요소 4개를 쭉 배치하면 반응형작업을 해줄 때 애로사항이 있기 때문이죠. 그래서 left섹션과 right섹션을 나누고 그안에 컨텐츠를 적절히 배치하였습니다.
flex는 간단합니다. flex는 자신의 자식요소에 적용할 수 있기 때문에, 부모에 display : flex를 주고 flex-wrap : wrap을 이용해 레이아웃을 배치해줍니다. 

 * {
    margin: 0;
    padding: 0;
}

#wrap {}

#header {
    height: 100px;
    background-color: #EEEBE9;
}

#nav {
    height: 100px;
    background-color: #B9AAA5;
}

#main {
    height: 780px;
    background-color: #886F65;
}

#footer {
    height: 100px;
    background-color: #4E342E;
}

.container {
    width: 1200px;
    height: inherit;
    background-color: rgba(0, 0, 0, 0.3);
    margin: 0 auto;
}

.contents .left {}

.contents .left .cont1 {
    width: 100%;
    height: 100px;
    background-color: #74574A;
}

.contents .right {
    display: flex;
    flex-wrap: wrap;
}

.contents .right .cont2 {
    width: 100%;
    height: 200px;
    background-color: #684D43;
}

.contents .right .cont3 {
    width: 50%;
    height: 480px;
    background-color: #594139;
}

.contents .right .cont4 {
    width: 50%;
    height: 480px;
    background-color: #4A352F;
}

/* 미디어 쿼리 */
@media(max-width : 1220px) {
    .container {
        width: 96%;
    }

    .contents {
        display: flex;
        flex-wrap: wrap;
    }

    .contents .left {
        width: 30%;
    }

    .contents .left .cont1 {
        width: 100%;
        height: 780px;
        background-color: #74574A;
    }

    .contents .right {
        width: 70%;
        display: flex;
        flex-wrap: wrap;
        height: 780px;
    }

    .contents .right .cont2 {
        width: 100%;
        height: 390px;
    }

    .contents .right .cont3 {
        width: 50%;
        height: 390px;
    }

    .contents .right .cont4 {
        width: 50%;
        height: 390px;
    }
}

@media(max-width : 768px) {

    }

grid를 이용한 레이아웃

grid를 이용한 레이아웃 방법입니다.
grid는 2차원 레이아웃 형태를 구성하기에 좋은 방법입니다.
grid는 flex와 마찬가지로 부모요소가 자식요소의 모든걸 컨트롤 합니다. 따라서 레이아웃에 따라 적절한 넓이와 높이값을 지정하고, 테이블이라 생각하고 레이아웃을 구성하면 깔끔하게 레이아웃이 구성됩니다. 

* {
    margin: 0;
}

#wrap {}

#header {
    height: 100px;
    background-color: #EEEBE9;
}

#nav {
    height: 100px;
    background-color: #B9AAA5;
}

#main {
    height: 780px;
    background-color: #886F65;
}

#footer {
    height: 100px;
    background-color: #4E342E;
}

.container {
    width: 1200px;
    height: inherit;
    margin: 0 auto;
    background-color: rgba(0, 0, 0, 0.3);
}

.contents {
    display: grid;
    grid-template-areas:
        "cont1 cont1"
        "cont2 cont2"
        "cont3 cont4"
    ;
    grid-template-columns: 50% 50%;
    grid-template-rows: 100px 200px 480px;
}

.contents .cont1 {
    background-color: #74574A;
    grid-area: cont1;
}

.contents .cont2 {
    background-color: #684D43;
    grid-area: cont2;
}

.contents .cont3 {
    background-color: #594139;
    grid-area: cont3;
}

.contents .cont4 {
    background-color: #4A352F;
    grid-area: cont4;
}

@media(max-width:1220px) {
    .container {
        width: 96%;
    }

    .contents {
        grid-template-areas:
            "cont1 cont2 cont2"
            "cont1 cont3 cont4"
        ;
        grid-template-columns: repeat(3,1fr);
        /* fr반응형 3등분 했습니다. */
        /* repeat(3, 33.3333%) */
        /* 다 같은 얘기입니다. */
        grid-template-rows: repeat(2, 390px);
    }

}

@media(max-width:768px) {
    .container {
        width: 100%;
    }
    .contents {
        grid-template-areas:
            "cont1 cont2"
            "cont1 cont3"
            "cont1 cont4"
        ;
        grid-template-columns: 30% 70%;
        grid-template-rows: repeat(3,1fr);
    }
}

@media(max-width:480px) {
    #header .container{
        width: 100%;
    }
    #nav .container{
        width: 100%;
    }
    #footer .container{
        width: 100%;
    }
    .contents {
        grid-template-areas:
            "cont1"
            "cont2"
            "cont3"
            "cont4"
        ;
        grid-template-columns: 100%;
        grid-template-rows: 150px 210px 210px 210px;
    }
}
