레이아웃03

이번 레이아웃은 저번 시간에 배웠던 레이아웃보다 한 단계 진화해서, 순서대로 header, nav, (aside, section(part1, part2, part3)), footer로 구성되어 있습니다.
다음은 레이아웃 구성형태 입니다.

01. float을 이용한 레이아웃

aside 영역과 section 영역을 한줄에 배치하기 위해서 float속성을 사용하겠습니다. section의 자식들은 부모에 float를 주면 알아서 배치됩니다. 하지만 float의 영향 때문에 밑의 footer영역이 없어지기 때문에 clear:both 속성을 사용해서 float에서 자유롭게 해줍니다. 

* {
    margin: 0;
}

body {
    background-color: #E1F5FE;
}

#wrap {
    width: 1200px;
    margin: 0 auto;
}

#header {
    width: 100%;
    height: 100px;
    background-color: #B3E5FC;
}

#nav {
    width: 100%;
    height: 100px;
    background-color: #81D4FA;
}

#aside {
    width: 25%;
    height: 780px;
    background-color: #4FC3F7;
    float: left;
}

#section_a {
    width: 75%;
    background-color: #29B6F6;
    height: 260px;
    float: left;
}

#section_b {
    width: 75%;
    background-color: #03A9F4;
    height: 260px;
    float: left;
}

#section_c {
    width: 75%;
    background-color: #039BE5;
    height: 260px;
    float: left;
}

#footer {
    width: 100%;
    height: 100px;
    background-color: #0288D1;
    clear: both;
}

@media(max-width:1220px) {
    #wrap {
        width: 96%;
    }

    #section_a{
        height: 260px;
    }
    #section_b {
        width: 37.5%;
        height: 520px;
    }

    #section_c {
        width: 37.5%;
        height: 520px;
    }
}

@media(max-width:768px) {
    #wrap {
        width: 100%;
    }
    #aside{
        width: 33%;
    }
    #section_a{
        width: 67%;
        height: 390px;
    }
    #section_b {
        width: 67%;
        height: 390px;
    }

    #section_c {
        display: none;
    }
}

@media(max-width:480px) {
    #aside {
        width: 100%;
        height: 300px;
    }

    #section_a {
        width: 100%;
        height: 200px;
    }
    #section_b {
        width: 100%;
        height: 430px;
    }
    #section_c {
        width: 100%;
        height: 150px;
    }
}

02. flex를 이용한 레이아웃

flex속성은 모든 태그의 부모인 wrap에 display:flex를 지정 후, 요소들이 잘 배치될 수 있도록 flex-wrap : wrap;를 사용해주면 간단하게 레이아웃이 완성됩니다.

    margin: 0;
}

body {
    background-color: #E1F5FE;
}

#wrap {
    width: 1200px;
    margin: 0 auto;
    display: flex;
    flex-wrap: wrap;
}

.container{
    display: flex;
    flex-wrap: wrap;
    width: 75%;
}
#header {
    width: 100%;
    height: 100px;
    background-color: #B3E5FC;
}

#nav {
    width: 100%;
    height: 100px;
    background-color: #81D4FA;
}

#aside {
    width: 25%;
    height: 780px;

}

#section_a {
    width: 100%;
    background-color: #29B6F6;

}

#section_b {
    width: 100%;
    background-color: #03A9F4;

#section_c {

03. grid를 이용한 레이아웃

grid속성은 모든 태그들을 감싸는 부모태그에서 자식요소의 모든 것을 컨트롤 하는 것 입니다. 부모태그에서 grid-templates-area로 영역을 지정 후 grid-templates-row & column으로 열너비와 행높이를 지정합니다. 또한, 각각 자식태그에서 grid-templates-area로정의한 영역의 이름을 gird-area로 정의해 주어야 합니다. 

* {
    margin: 0;
}

body {
    background-color: #E1F5FE;
}

#wrap {
    width: 1200px;
    margin: 0 auto;
    display: grid;
    grid-template-areas:
        "header header"
        "nav nav"
        "aside section_a"
        "aside section_b"
        "aside section_c"
        "footer footer"
    ;
    grid-template-rows: 100px 100px 260px 260px 260px 100px;
    grid-template-columns: 25% 75%;
}

#header {
    background-color: #B3E5FC;
    grid-area: header;
}

#nav {
    background-color: #81D4FA;
    grid-area: nav;
}

#aside {
    background-color: #4FC3F7;
    grid-area: aside;
}

#section_a {
    background-color: #29B6F6;
    grid-area: section_a;
}

#section_b {
    background-color: #03A9F4;
    grid-area: section_b;
}

#section_c {
    background-color: #039BE5;
    grid-area: section_c;
}

#footer {
    background-color: #0288D1;
    grid-area: footer;
}

@media(max-width:1220px) {
    #wrap {
        width: 96%;
        margin: 0 auto;
        display: grid;
        grid-template-areas:
            "header header header"
            "nav nav nav"
            "aside section_a section_a"
            "aside section_b section_c"
            "footer footer footer"
        ;
        grid-template-rows: 100px 100px 260px 520px 100px;
        grid-template-columns: 25% 37.5% 37.5%;
    }
}

@media(max-width:768px) {
    #wrap {
        width: 100%;
        margin: 0 auto;
        display: grid;
        grid-template-areas:
            "header header"
            "nav nav"
            "aside section_a"
            "aside section_b"
            "footer footer"
        ;
        grid-template-rows: 100px 100px 520px 520px 100px;
        grid-template-columns: 25% 75%;
    }
}

@media(max-width:480px) {
    #wrap {
        width: 100%;
        margin: 0 auto;
        display: grid;
        grid-template-areas:
            "header header"
            "nav nav"
            "aside aside"
            "section_a section_a"
            "section_b section_b"
            "section_c section_c"
            "footer footer"
        ;
        grid-template-rows: 100px 100px 200px 430px 150px 100px;
        grid-template-columns: 25% 75%;
    }
}
